gofus [![Build Status](https://drone.io/github.com/Blackrush/gofus/status.png)](https://drone.io/github.com/Blackrush/gofus/latest)
=====

Gofus is a Dofus 1.29 emulator written in Go.

### Installation

```bash
$ go get github.com/Blackrush/gofus
```

### Usage

```bash
$ gofus -h # print usage
```
